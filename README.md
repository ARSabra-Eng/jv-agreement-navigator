# JV Agreement Navigator

A self-contained, offline-first preparation workspace for pre-tender and post-tender Joint Venture / Consortium agreements.

## Open the website

Open `index.html` in a modern desktop browser. No installation, account, server, or internet connection is required.

For stricter browser environments, run any local static server in this folder, for example:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Included functions

- Separate pre-tender and post-tender workflows
- Structured project, party, bid, governance, contribution, execution, commercial, and risk forms
- Readiness scoring and dynamic risk flags
- Pre-tender and post-tender preparation checklists
- Searchable clause explanation library
- Automatic local draft saving through browser storage
- Word-compatible summary, CSV checklist / risk register, JSON backup, and print / PDF outputs
- JSON backup restoration
- Responsive desktop, tablet, and mobile layouts

All calculations and downloads are created locally in the browser. The website does not call any external service.

## Professional use note

The application is a preparation and decision-recording aid. Final JV agreements should be reviewed against the tender or main contract, governing law, corporate approvals, licensing, tax, competition, security, and execution formalities.
